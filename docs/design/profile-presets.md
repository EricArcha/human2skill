# Profile Presets

## 1. 共同内核

每类人物 profile 都必须提炼以下共同内核：

- 核心思维模型。
- 决策启发式。
- 表达 DNA。
- 压力和冲突反应。
- 价值排序。
- 反模式。
- 诚实边界。
- 证据和置信度摘要。

差异只体现在权重、访谈问题和输出专项层。

## 2. 同事/合作者

适用对象：

- 同事、上级、下级、客户、合作伙伴、项目协作者。

专项权重：

| 维度 | 权重 |
| --- | --- |
| 工作方法 | 高 |
| 决策标准 | 高 |
| 表达 DNA | 中高 |
| 人际互动 | 中 |
| 情绪支持 | 低 |

重点问题：

- 他接到需求后第一步做什么？
- 他如何判断一个方案值不值得做？
- 他如何做 review、反馈或拒绝？
- 他压力大时会主动沟通还是消失？
- 哪些事情他一定会推掉？

输出专项层：

- 职责和边界。
- 工作流。
- 输出偏好。
- 协作规则。
- 风险和反模式。

## 3. 朋友/亲人

适用对象：

- 朋友、伴侣、父母、子女、亲密关系对象。

专项权重：

| 维度 | 权重 |
| --- | --- |
| 关系互动 | 高 |
| 情绪反应 | 高 |
| 表达 DNA | 高 |
| 价值排序 | 中高 |
| 工作方法 | 低 |

重点问题：

- 他表达关心的方式是什么？
- 他遇到冲突时是解释、回避、反击还是沉默？
- 他什么情况下会主动靠近或疏远？
- 他对边界、责任和亏欠怎么看？
- 你最容易误解他的地方是什么？

输出专项层：

- 关系互动模式。
- 情绪触发点。
- 支持和拒绝方式。
- 冲突修复方式。
- 亲密边界。

## 4. 导师/专家

适用对象：

- 老师、导师、顾问、行业专家、创作者、长期影响用户的人。

专项权重：

| 维度 | 权重 |
| --- | --- |
| 观点体系 | 高 |
| 判断框架 | 高 |
| 教学反馈 | 中高 |
| 表达 DNA | 中高 |
| 关系互动 | 中 |

重点问题：

- 他最常用什么框架解释问题？
- 他如何评价一个人的进步或错误？
- 他在什么问题上非常坚持？
- 他如何把复杂问题讲简单？
- 他承认不确定时会怎么说？

输出专项层：

- 核心观点体系。
- 教学和反馈方式。
- 判断标准。
- 典型问题处理。
- 盲区和适用边界。

## 5. 自己

适用对象：

- 用户本人、过去的自己、未来理想自我。

专项权重：

| 维度 | 权重 |
| --- | --- |
| 自我反思 | 高 |
| 决策镜像 | 高 |
| 表达 DNA | 中 |
| 长期偏好 | 高 |
| 关系互动 | 中 |

重点问题：

- 你过去反复做出什么类型的选择？
- 你在什么场景下最容易后悔？
- 你真实重视什么，但经常嘴上不承认？
- 哪些决定符合长期的你，哪些只是短期焦虑？
- 未来的你希望现在的你少做什么？

输出专项层：

- 自我决策镜像。
- 长期价值排序。
- 反复出现的盲区。
- 行动建议的过滤规则。
- 与未来自我的对话协议。

## 6. Profile 选择规则

用户明确选择时按用户选择。

用户未明确选择时：

- 出现公司、职级、项目、评审、交付、协作，默认同事/合作者。
- 出现朋友、父母、伴侣、孩子、关系、情绪，默认朋友/亲人。
- 出现老师、导师、专家、博主、课程、方法论，默认导师/专家。
- 出现“我自己”“过去的我”“未来的我”，默认自己。

若人物跨 profile，选择一个主 profile，并记录副 profile。例如“创业合伙人”可主选同事/合作者，副选朋友/亲人。
