# human2skill — Claude Code Meta-Skill

Distill a real person into a reusable perspective advisor skill.

## Install

Copy this directory into your Claude Code skills directory:

```bash
cp -r . ~/.claude/skills/human2skill
```

Then type **human2skill** in Claude Code to start.
