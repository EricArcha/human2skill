# human2skill — OpenClaw Meta-Skill

Distill a real person into a reusable perspective advisor skill.

## Install

Copy this directory into your OpenClaw skills directory:

```bash
cp -r . ~/.openclaw/skills/human2skill
```

Then type **human2skill** in OpenClaw to start.
